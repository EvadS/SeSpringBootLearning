

## Git config 

Глобальные настройки для логгирования 
все созраняется HOME_DIR/.gitconfig

```bash
  git config  --global user.name
```

```bash
  git config  --global user.email
```
все конфигурации 
```bash
  git config  -l
```

установить для одного репо (локально)
```bash
git config  --global user.name
```
